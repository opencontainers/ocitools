# NAME
oci-runtime-tool-runtime-validate - Validate an OCI runtime

# SYNOPSIS
**oci-runtime-tool runtime-validate**  *[OPTIONS]*

# DESCRIPTION

Validate an OCI runtime

# OPTIONS
**--help**
  Print usage statement

**--runtime**=""
  OCI runtime. The default is *runc*.
